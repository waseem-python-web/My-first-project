# Grade Calculator
This is a simple Python project that takes student name and marks and calculates grade.


## Features
1)  User input (name + marks)
2) Automatic grade calculation
3) Clean output

## How to Run
python main.py

## Output

===== Student Grade Calculator =====

Enter your name:waseem
Enter your marks (0-100): 67
=====Your Result =====
Your Name :waseem
Your Marks:67.0
YOUR Grade:C
